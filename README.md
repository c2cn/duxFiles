# 简介

DuxFile 是一款PHP多存储驱动的文件管理类，支持多种云存储平台，使用 `guzzle6` 做驱动请求，摒弃各类 SDK 的臃肿，统一使用调用方便各类框架、系统使用。

# 支持平台

- 本地服务器
- 阿里云 OSS
- 腾讯云 COS
- 七牛云存储
- 又拍云存储

# 环境支持

- 语言版本：PHP 7.1+
